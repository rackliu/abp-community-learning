# 章節學習目標與先修需求

參考：[`abp-community-learning-TOC.md`](abp-community-learning-TOC.md:2)

## Ch1 現代軟體開發與 ABP Framework
- 學習目標：理解現代軟體開發趨勢、ABP 架構理念與社群生態；能評估單層/分層/微服務選型。
- 先修知識：基本軟體工程概念、C# 與 ASP.NET Core 基礎。
- 建議時數：2 小時

## Ch2 ABP Framework 快速入門
- 學習目標：使用 CLI/Visual Studio 建立專案、啟動範例應用、理解 Module 啟動流程。
- 先修知識：C# 與 .NET 8/9 基礎、Visual Studio 或 VS Code 操作。
- 建議時數：4 小時

## Ch3 實戰應用程式開發流程
- 學習目標：掌握從需求到交付的開發流程，能繪製與解讀專案流程圖（Mermaid）。
- 先修知識：基本專案管理與版本控制概念。
- 建議時數：3 小時

## Ch4 認識官方參考解決方案
- 學習目標：熟悉官方範例與社群套件，能挑選適合的參考實作。
- 先修知識：已完成 Ch1-Ch3 或同等認知。
- 建議時數：2 小時

## Ch5 ASP.NET Core 與 ABP 架構深入解析
- 學習目標：理解 Startup/Module、依賴注入與 ABP 模組化設計與擴充點。
- 先修知識：ASP.NET Core 中介軟體、DI 概念、基本 ABP 專案經驗。
- 建議時數：6 小時

## Ch6 資料存取基礎設施
- 學習目標：掌握 Entity/Repository/UoW 模式、EF Core 與 MongoDB 在 ABP 的整合。
- 先修知識：關聯式資料庫、EF Core 基礎、LINQ。
- 建議時數：8 小時

## Ch7 橫切關注 (Cross-Cutting Concerns)
- 學習目標：實作權限管理、驗證、例外處理、日誌與審核追蹤等橫切功能。
- 先修知識：安全基礎（JWT/OAuth 基本概念）、Logging 技術。
- 建議時數：6 小時

## Ch8 開源特色與服務功能
- 學習目標：使用者管理、Auto API、資料篩選、快取與本地化配置與實作。
- 先修知識：ABP 自動 API 機制、快取概念。
- 建議時數：6 小時

## Ch9 領域驅動設計理論（DDD）
- 學習目標：理解 DDD 基本概念：聚合、實體、值物件、邊界上下文。
- 先修知識：面向物件設計基礎、UML 或建模概念。
- 建議時數：4 小時

## Ch10 領域層實作
- 學習目標：實作聚合根、領域事件與領域服務，將商業邏輯封裝於領域層。
- 先修知識：Ch9 或 DDD 基礎經驗、C# 類別設計能力。
- 建議時數：4 小時

## Ch11 應用層設計
- 學習目標：設計 DTO、應用服務、授權策略，並使用 Mapster 等工具進行對映。
- 先修知識：熟悉 Controller/Service 層、序列化與 DTO 概念。
- 建議時數：4 小時

## Ch12 MVC/Razor Pages 前端開發
- 學習目標：掌握 Razor 頁面、TagHelper、分頁設計與資產管理（Bundling/NPM）。
- 先修知識：HTML/CSS/基本 JavaScript、Razor 基礎。
- 建議時數：6 小時

## Ch13 Blazor WebAssembly UI（社群版）
- 學習目標：了解 Blazor 組件模型、Server vs WASM 的差異以及可用開源解法。
- 先修知識：C# 在前端的運用、基本前端概念。
- 建議時數：4 小時

## Ch14 微服務架構設計
- 學習目標：設計微服務邊界、整合通訊、選擇範本並實作 Integration Service。
- 先修知識：分散式系統基礎、REST/gRPC 概念。
- 建議時數：6 小時

## Ch15 模組化開發進階
- 學習目標：設計 Application Module、套件化、發布 NuGet 與開源包管理流程。
- 先修知識：NuGet 基礎、模組化設計經驗。
- 建議時數：4 小時

## Ch16 多租戶系統（社群版）
- 學習目標：配置多租戶架構、資料隔離策略與租戶切換、功能管理（Feature）。
- 先修知識：資料隔離概念、ABP 多租戶基礎。
- 建議時數：4 小時

## Ch17 ABP 測試架構介紹
- 學習目標：建立單元測試、整合測試範例，理解測試基礎設施與測試資料準備。
- 先修知識：xUnit/MSTest/NUnit 使用經驗、Mocking 概念。
- 建議時數：6 小時

## Ch18 快取策略與效能優化
- 學習目標：設計 Redis 快取策略、識別 N+1 問題並優化查詢效能。
- 先修知識：資料庫查詢原理、性能分析工具基礎。
- 建議時數：4 小時

## Ch19 安全授權與資料保護
- 學習目標：建立權限管理、資料加密、隱私保護與 GDPR 對應建議。
- 先修知識：安全概念（OAuth/JWT）、加密/Hash 原理。
- 建議時數：4 小時

## Ch20 React 基礎 UI 開發
- 學習目標：將 React 前端與 ABP API 整合、使用開源主題與組件。
- 先修知識：React 基礎、前端工具鏈 (Webpack/Vite/NPM)。
- 建議時數：4 小時

## Ch21 LeptonX Lite / AdminLTE 主題（社群版）
- 學習目標：客製 LeptonX Lite（或類似社群主題）、配置深/淺色模式與 CSS 變數。
- 先修知識：CSS/SASS、前端主題配置經驗。
- 建議時數：3 小時

## Ch22 容器化與 Kubernetes 部署
- 學習目標：Dockerize 應用、建立 Helm chart、在 K8s 上部署與監控。
- 先修知識：Docker 基礎、Kubernetes 概念。
- 建議時數：6 小時

## Ch23 升級策略與遷移指引
- 學習目標：制定開源版升級流程、處理遷移衝突與從 ASP.NET Boilerplate 遷移至 ABP 的實務步驟。
- 先修知識：資料庫遷移、版本管理、過去專案經驗。
- 建議時數：4 小時

## Ch24 社群熱門開源模組整合
- 學習目標：整合 EasyAbp 等社群模組、評估相容性與客製化策略。
- 先修知識：套件整合經驗、NuGet 與 Git 操作。
- 建議時數：4 小時

## Ch25 完整專案架構範例與教學
- 學習目標：設計並實作一個完整示範專案（企業/ERP/SaaS 可行範例，依社群版限制）。
- 先修知識：已完成多數前述章節，具備整合能力。
- 建議時數：8-12 小時
